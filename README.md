# 🔵BB 3D空间提示词

一个 ComfyUI 插件，将交互式 3D 查看器嵌入到节点中。该节点会出现在 ComfyUI 菜单目录：`🔵BB 3DViewer`。

## 功能特性

- 嵌入式3D查看器，支持鼠标交互控制
- 实时相机参数调节（位置X/Y/Z，FOV）
- 支持图像输入并显示在3D场景中
- 基于Three.js的完整3D环境

## 安装

1. 将`ComfyUI_3DViewer`文件夹复制到ComfyUI的`custom_nodes`目录中
2. 重启ComfyUI

## 使用方法

1. 在ComfyUI工作区中搜索"🔵BB 3D空间提示词"节点（或在菜单目录 `🔵BB 3DViewer` 下查找）
2. 添加节点后，会看到一个嵌入的3D查看器
3. 使用鼠标控制3D场景：
   - 左键拖拽：旋转视角
   - 滚轮：缩放
   - 右键拖拽：平移视图
4. 调节滑块控件来精确控制相机参数
5. 可选：连接图像输入，图片会显示在3D场景的小人模型正面

## 鼠标控制说明

- **左键拖拽**: 围绕目标点旋转视角，支持360°全方位旋转
- **鼠标滚轮**: 放大/缩小视图，缩放范围限制在2-20单位距离内
- **右键拖拽**: 平移视图位置，可在不改变旋转角度的情况下移动观察点
- **右键菜单**: 仅在执行平移操作时会被禁用，普通右键点击仍可弹出浏览器菜单
